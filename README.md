# LOL Worlds Simulator

一个纯文字驱动的英雄联盟世界赛模拟器。

玩家只需要不断点击“下一步”，观看瑞士轮、双败淘汰赛、总决赛的完整推演，并见证冠军诞生。

## 当前版本

V1.0 正式版

## 核心玩法

- 16 支队伍参加世界赛。
- 瑞士轮阶段：3 胜晋级，3 负淘汰。
- 淘汰赛阶段：8 强双败赛制。
- 总决赛：BO7，胜者组冠军默认 1-0 领先。
- 所有比赛由战力、波动、气势和随机数决定。
- 玩家不需要操作队伍，只需要推进比赛流程。

## 已实现功能

- 瑞士轮抽签与对阵图
- 瑞士轮 BO1 / BO3 规则
- 双败淘汰赛对阵图
- BO5 小局逐局模拟
- 总决赛 BO7
- 胜者组冠军 1-0 领先
- 每场比赛随机数开奖
- 胜率能量条
- 队伍战力、波动、气势系统
- 快速模拟瑞士轮 / 淘汰赛
- 自定义赛事
- 修改默认队伍数据
- 永久数据统计
- 冠军榜
- 冠军之路回顾
- 清空数据二次确认


## 设计原则

这个模拟器刻意保持简单：

- 不做选手系统
- 不做 BP 系统
- 不做英雄系统
- 不做装备系统
- 不做转会市场
- 不做自动剧情新闻

它的核心体验就是：

```text
点击下一步
观看随机数开奖
见证冠军诞生
```
